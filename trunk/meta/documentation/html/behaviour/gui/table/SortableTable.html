getText

Object getText(entity, text)
    Returns either the value of the entity property using text as a key or text itself. So getText({name: "alex"}, "name") would return "alex" but getText({name: "alex"}, "age") would return "age". If entityID has . characters in it, this method will step down into entity to find the key, so getText({name: "alex", innerObject : {name: "bob"}}, "innerObject.bob") would return "bob"
    Parameters:
      Object - entity
      String - text
